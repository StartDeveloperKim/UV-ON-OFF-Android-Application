# ICT 융합 공모전 프로젝트
해당 레포는 앱 프로젝트 전용 레포지토리입니다.
### 주제
UV-C를 활용한 차량 내부 소독
### 개발환경
- 하드웨어
	- 라즈베리파이, PIR센서, LED
	- C, MQTT 프로토콜
- 어플리케이션
	- 안드로이드 스튜디오
	- MQTT 프로토콜
### 애플리케이션 개요
차량 내부에서 UV가 켜지거나 꺼지면 알람이 오고 원격으로 ON/OFF를 동작하게 할 수 있다.
### 애플리케이션 요구사항
- UV 라이트 ON/OFF 알림
- 수동으로 UV 라이트를 ON/OFF 할 수 있도록 한다.
- 프로토콜은 MQTT를 사용한다.
#### MQTT프로토콜을 선택한 이유
MQTT는 IoT에서 저전력으로 사용할 수 있는 프로토콜로서 센서정보 수집에 유리하기에 해당 프로토콜을 통신 프로토콜로서 선정하였다.
